HASHES (WINDOWS)
================ 

7-Zip:
------
 
7-zip -> right click on the file + CRC SHA (CRC-32, CRC-64, SHA1, SHA256 and BLAKE2sp) 

or

7z h [-scrc{Method}] [files] 

Supported methods: CRC32, CRC64, SHA1, SHA256, BLAKE2sp. Default method is CRC32. 
(ex: 7z h -scrcSHA256 files) 
 
Certutil: 
---------

certutil -hashfile -? (Hash algorithms: MD2 MD4 MD5 SHA1 SHA256 SHA384 SHA512) 
(ex: certutil -hashfile file SHA256)

PowerShell Get-FileHash:
------------------------

Get-FileHash [-LiteralPath] <String[]> [[-Algorithm] <String>] [<CommonParameters>]
The acceptable values for this parameter are SHA1, SHA256, SHA384, SHA512 and MD5

More info: https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.utility/get-filehash?view=powershell-7.1

Example:

Get-FileHash order.ps -Algorith MD5 | Format-List

Algorithm : MD5
Hash      : A25F7F0B29EE0B3968C860738533A4B9
Path      : C:\users\varanda\Desktop\order.ps
